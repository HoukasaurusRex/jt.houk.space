## Team

### Ignite Talks

Researched and presented on MySQL geospatial data types and querying. This ties directly into the KML delivery zones feature, where we'll need to store polygon boundaries and run point-in-polygon queries to determine whether a customer's address falls within a dispensary's delivery area.

### KML Delivery Zones

Discussed the functionality and scope of the KML delivery zone feature with the team. The main questions center on how we ingest KML files and validate addresses at checkout time.

## Projects

### Inventory Snapshots

Finished performance testing the snapshot queries in sandbox and completed the resync changes, which are now in code review. The resync feature lets retailers reconcile their snapshot data against actual inventory, which is essential for audit preparation.
