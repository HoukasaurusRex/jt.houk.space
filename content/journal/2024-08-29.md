## Investigations

Investigated why OCS submissions weren't matching inventory movements for Ganjika House. The Ontario Cannabis Store requires retailers to report inventory changes accurately, so discrepancies between what's submitted and what actually moved in the system need to be reconciled before the regulator flags them.

## Projects

### Inventory Snapshots

Finished the code for resyncing snapshots, demoed the changes in a feature branch, and tested performance in sandbox. The resync operation compares the snapshot state against live inventory and corrects any drift, which retailers need before submitting compliance reports.
